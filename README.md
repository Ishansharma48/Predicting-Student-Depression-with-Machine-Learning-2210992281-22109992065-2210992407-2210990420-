# 🎯 Project Objective

This project focuses on predicting student depression using Machine Learning techniques by analyzing academic, behavioral, psychological, and lifestyle-related factors. The system helps identify students who may be at risk of depression and enables early intervention and mental health support.

---

# 📊 Dataset Information

- Dataset Source: Kaggle Student Depression Dataset
- Total Records: 27,902
- Total Features: 18
- Target Variable:
  - `1` → Depressed
  - `0` → Not Depressed

### Features Used
- Age
- Gender
- CGPA
- Sleep Duration
- Academic Pressure
- Work Pressure
- Study Satisfaction
- Job Satisfaction
- Dietary Habits
- Financial Status
- Work/Study Hours
- Family History of Mental Illness
- Suicidal Thoughts
- Depression Status

# 📈 Results

| Classifier | Accuracy |
|------------|-----------|
| Logistic Regression | 86.15% |
| SVM | 86.09% |
| Random Forest | 85.20% |
| AdaBoost | 85.16% |
| ANN | 81.90% |

### Best Performing Model
✅ Logistic Regression achieved the highest accuracy of **86.15%**
